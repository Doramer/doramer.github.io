提权

suid提权

原理：

能够执行命令的shell命令被赋予了suid权限

例如find被赋予suid权限

find zhansan -exec whoami \;
查找zhangsan文件夹的时候可以执行whoami命令

利用：

shell 进入shell交互

find / -perm -u=s -type f 2>/dev/null

解释参数

find / -perm -u=s -type f 2>/dev/null
/表示从文件系统的顶部（根）开始并找到每个目录
-perm 表示搜索随后的权限
-u = s表示查找root用户拥有的文件
-type表示我们正在寻找的文件类型
f 表示常规文件，而不是目录或特殊文件
2表示该进程的第二个文件描述符，即stderr（标准错误）
>表示重定向
/dev/null是一个特殊的文件系统对象，它将丢弃写入其中的所有内容。
